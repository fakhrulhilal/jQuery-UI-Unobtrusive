jQuery UI Unobtrusive

It's an implementation of creating jQuery UI widget with unobtrusive style (attribute based declaration). You don't need to write the code, just set up via HTML attributes and you're done.

Requirement

  1. jQuery
  2. jQuery UI
  3. jQuery Utility

Getting started

  1. Download the requirements above
  2. Include in your page (put in the last bottom HTML code is better)
  3. Write your widget

Note about this project

Convention used: 1. This project only parse with these convention syntax:

<div data-role="jqueryui-_widgetname_" />
  1. Set the option using HTML5 data attribute: <div data-role="jqueryui-_widgetname_" data-_widgetname_-_optionName_="_optionValue_" /> For optionName please refer to official jQuery UI documentation

Currently we've support these widgets: 1. jQuery UI dialog
